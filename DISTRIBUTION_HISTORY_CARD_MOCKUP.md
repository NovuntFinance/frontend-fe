## Color Coding

### **Status Icons:**

- **Blue Background + Blue Checkmark**: Performance Pool, Qualified
- **Emerald Background + Green Checkmark**: Premium Pool, Qualified
- **Red Background + Red X**: Not Qualified (or Gray if not qualified)

### **Text Colors:**

- **White/Bold**: Pool type name
- **Emerald Green**: Earnings amount (+$X.XX)
- **Muted Gray**: Period, Share, Date
- **White**: Rank badge text
- **Emerald**: Qualified badge

### **Card Theme:**

- **Purple Theme**: Card uses purple color theme (from `colorTheme="purple"`)

---

## Responsive Behavior

### **Desktop View:**

- Distribution items display in full width
- Amount and date right-aligned
- Filter buttons in a row

### **Mobile View:**

- Distribution items stack vertically
- Amount and date may stack below pool info
- Filter buttons may wrap to multiple rows

---

## Interactive Elements

1. **Filter Buttons**: Click to filter distributions by type
2. **Distribution Items**: Hover effect (background lightens)
3. **Pagination Buttons**: Click to navigate between pages
4. **Empty State Button**: Click to scroll to rank requirements
